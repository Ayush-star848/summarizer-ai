# Summarizer AI

AI-powered content summarizer built with:

- LangChain
- Mistral AI
- CustomTkinter
- Python

## Features

- Movie Summarization
- Article Summarization
- Blog Summarization
- Modern Desktop UI

## Installation

```bash
git clone <repo-url>
cd Summarizer

uv venv
.venv\Scripts\activate

uv pip install -r requirements.txt
```

Create a `.env` file:

```env
MISTRAL_API_KEY=your_api_key
```

Run:

```bash
uv run ui.py
```
